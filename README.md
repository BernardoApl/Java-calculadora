# Java-calculadora
Calculadora em java | simples

## Estrutura sugerida (CODIGO/)

- `CODIGO/backend/`: código Java (backend) no layout padrão `src/main/java`
- `CODIGO/backend/src/test/java`: testes unitários (auto-contidos)

### Como rodar os testes

Veja `CODIGO/backend/README.md` para compilar e rodar os testes unitários com `javac/java`.

Obs: `mvn test` só funciona se você tiver Maven instalado e baixar dependências.
